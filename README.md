# Learn Korean with Stray Kids Lyrics

**[🔗 PLAY AND LEARN HERE (Live Website)](https://wayindazen.github.io/learnkoreanwithstraykidslyrics/)**

An interactive web application designed for STAYs (and Korean learners in general) to learn Korean vocabulary, grammar, and sentence structures naturally through **Stray Kids** songs! 

## ✨ Features
* **Interactive Lyrics:** Click on words in the lyrics to see their meaning, dictionary form, and grammar rules.
* **Grammar & Lexicon Stickers:** A visual "sticker board" where definitions pop up. 
* **Cross-Lyrics Search:** Click on a word to see how Stray Kids used it in *other* songs!
* **Global & Song-Specific Quizzes:** Test your knowledge. Translate words from Korean to English or vice versa.
* **Grammar Cheat Sheets:** Quick references for Korean conjugation (Panmal, Polite, Formal, etc.).
* **Mobile Friendly:** Study on the go!

## 🚀 How to Use
1. Choose an album and a song from the main page.
2. Read the lyrics. If you don't know a word (highlighted in green/pink), click on it!
3. A sticker will appear with the translation and grammar explanation.
4. Ready to test yourself? Click the **Quiz** button to check what you've learned.

## 🛠️ Tech Stack
This project is built using pure Frontend technologies:
* **HTML5 & CSS3** (Fully responsive design, custom UI/UX)
* **Vanilla JavaScript** (DOM manipulation, quiz logic, search algorithms)
* **JSON** (Database for songs, lyrics, and grammar rules)

## 💡 Motivation
Learning a language is hard, but music makes it easier. Stray Kids write their own deeply meaningful lyrics, making their discography a goldmine for Korean learners. This project aims to bridge the gap between enjoying K-pop and actively studying the Korean language.
